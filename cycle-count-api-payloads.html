Cycle Count API Payloads

This document details the JSON payload structures for retrieving cycle count data from the WMSPro system, based on the `cyclecounts` and `cyclecountitems` database tables.

Get All Cycle Counts

GET /api/warehouse-management/cyclecount/getAll

Retrieves a list of all cycle count tasks. The response is a JSON array where each object represents a single cycle count in a master-detail format.

Get Cycle Count by ID

GET /api/warehouse-management/cyclecount/getbyid/{id}

Retrieves a single cycle count task by its unique ID. The response is a single JSON object with the same master-detail structure as the elements in the `getAll` response.

URL Parameter

  • id (required): The unique identifier (integer) of the cycle count task. Example: /getbyid/1
