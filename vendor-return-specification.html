Technical Specification: Vendor Return Management

Architecture Decision: This module is built as a standalone service (RTV - Return to Vendor) independent of the Customer RMA system. This separation allows for specialized logic handling Debit Notes, Tax Reversals, and Batch-specific warehouse extraction that differ from consumer return workflows.

1. Database Schema (Independent RTV)

Dedicated tables to track outbound logistics for defective or overstock supplier items.

2. API Endpoints Specification

POST /api/v1/logistics/rtv/initiate

Initializes a new vendor return request. Validates stock availability in the specific warehouse before approval.

Request Payload

{
  "supplierId": 2044,
  "purchaseOrderId": 8812,
  "warehouseId": 7,
  "items": [
    {
      "productId": 101,
      "quantity": 15,
      "batchId": "B-2024-X9",
      "reason": "DMG"
    }
  ],
  "notes": "Returned due to physical packaging damage during transit"
}

Response Payload (201 Created)

{
  "success": true,
  "rtvId": 50012,
  "status": "Draft",
  "message": "Vendor return initiated. Stock marked as 'Restricted-Return'."
}
PUT /api/v1/logistics/rtv/{rtvId}/dispatch

Triggered when the goods physically leave the warehouse. This endpoint decrements the inventory ledger permanently.

Request Payload

{
  "trackingNumber": "TRK8827110",
  "transporterName": "BlueDart Logistics",
  "dispatchDocs": ["base64_string_of_gate_pass"]
}

Response Payload (200 OK)

{
  "success": true,
  "inventoryUpdated": true,
  "debitNoteGenerated": "DN-2025-001"
}
GET /api/v1/logistics/rtv/list?status=Approved

Fetches a filtered list of vendor returns for the logistics dashboard.

3. Operational Status Flow

1. Draft (Inspection)
2. Approved (Finance)
3. Shipped (Logistics)
4. Settled (Accounts)
  • Draft: Items are physically separated but still in system inventory (marked 'Restricted').
  • Approved: Finance has verified the Debit Note value.
  • Shipped: Gate pass issued. Inventory count is subtracted here.
  • Settled: Supplier acknowledged receipt and adjusted the outstanding balance.
